Bash For循环用法示例

Jan 5th, 2014 | Comments

来源：Bash For Loop Examples 编译：lingguo

1. for循环的第一种形式：

    for variable in 1 2 3 4 5
    do
        command
    done

示例1：

    #!/bin/bash
    for i in 1 2 3 4 5
    do
       echo "Welcome $i times"
    done

连续的值，可以通过{start..end}缩写，如1 2 3 4 5可以表示为：{1..5}，如： 示例2：

    #!/bin/bash
    for i in {1..5}
    do
       echo "Welcome $i times"
    done

bash 4.0及以上，支持设置变量的自增的步数，语法是：{start..end..increment}, 如： 示例3：

    #!/bin/bash
    echo "Bash version ${BASH_VERSION}..."
    for i in {0..10..2}
      do
         echo "Welcome $i times"
    done

使用for遍历文件，语法为：

    for file in file1 file2 file3 file4
    do
        commands
    done

示例4：

 #!/bin/bash
    for file in /etc/*
    do
        if [ "${file}" == "/etc/resolv.conf" ]
        then
            countNameservers=$(grep -c nameserver /etc/resolv.conf)
            echo "Total  ${countNameservers} nameservers defined in ${file}"
            break
       fi
    done

其中for file in /etc/* 等价于：

for file in `ls /etc/*`    
for file in $(ls /etc/*)

2. for循环的第二种形式：

与C语言for循环类似的语法：

for (( EXP1; EXP2; EXP3 ))
do
    commands
done

示例5：

 #!/bin/bash
    for (( c=1; c<=5; c++ ))
    do
       echo "Welcome $c times"
    done

当for的EXP1，EXP2，EXP3都为空时，即为无限循环： 示例6：

 #!/bin/bash
    for (( ; ; ))
    do
       echo "infinite loops [ hit CTRL+C to stop]"
    done

3. for循环也支持break和continue关键字，与C语言一致

示例7：

 #!/bin/bash
    for file in /etc/*
    do
        if [ "${file}" == "/etc/resolv.conf" ]
        then
            countNameservers=$(grep -c nameserver /etc/resolv.conf)
            echo "Total  ${countNameservers} nameservers defined in ${file}"
            break
       fi
    done

示例8：

 #!/bin/bash
    FILES="$@"
    for f in $FILES
    do
        # if .bak backup file exists, read next file
        if [ -f ${f}.bak ]
        then
            echo "Skiping $f file..."
            continue  # read next file and skip cp command
        fi
        # we are hear means no backup file exists, just use cp command to copy file
        /bin/cp $f $f.bak
    done
